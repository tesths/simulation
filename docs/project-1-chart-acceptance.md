## 人工验收

### 教师流程

1. 打开 `/teacher`
2. 输入教师密码登录
3. 进入 dashboard
4. 新建班级，设置组数
5. 页面显示新的学生专属链接

通过标准：

- 登录成功
- dashboard 可见
- 新班级创建成功
- 学生链接可复制和访问

### 学生流程

1. 打开老师生成的学生专属链接
2. 选择一个组
3. 进入填写页
4. 提交一组温度数据

通过标准：

- 能成功选组
- 能成功提交
- 提交后返回图表 JSON 正常

### 实时同步

1. 老师端保持打开 dashboard
2. 学生端再次提交一次

通过标准：

- 老师端对应组卡片自动刷新
- 不需要手动刷新整页
